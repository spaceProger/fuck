# 🖕 fuck

Утилита командной строки для управления фрустрацией и хаосом. Написана на Rust.

Вдохновлена Python-скриптом и сериалом «Кремниевая долина».

## Установка

```bash
cargo install --path .
```

## Использование

```
fuck [КОМАНДА]
```

### Основные команды

| Команда | Описание |
|---|---|
| `fuck it` | Генерирует профессиональное оправдание безделья |
| `fuck off` | Предлагает прекратить страдания |
| `fuck off --rude` | То же, но грубее |
| `fuck up` | Симулирует катастрофический сбой системы |
| `fuck around` | Бесполезный прогресс-бар, который иногда идёт назад |
| `fuck find-out` | Узнай, что бывает когда fuck around |
| `fuck this` | Анализирует текущий проект и ставит диагноз |
| `fuck why` | Философски объясняет, почему всё сломалось |
| `fuck count` | Счётчик запусков с достижениями |

### `fuck sv` — Кремниевая долина

| Команда | Описание |
|---|---|
| `fuck sv pied-piper [TEXT]` | Сжимает текст алгоритмом Middle-Out |
| `fuck sv not-hotdog [ITEM]` | Определяет, является ли объект хот-догом |
| `fuck sv hooli` | Цитата Гэвина Белсона |
| `fuck sv tabs-spaces` | Решает вечный спор раз и навсегда |
| `fuck sv nucleus [-p N] [-s N]` | Калькулятор Nucleus |
| `fuck sv erlich` | Монолог Эрлиха Бэкмана |
| `fuck sv standup` | Standup отчёт в стиле SV |
| `fuck sv valuation` | Оценка стартапа по науке |

### `fuck corp` — Корпоративный булшит

| Команда | Описание |
|---|---|
| `fuck corp meeting` | Повестка митинга, который мог быть письмом |
| `fuck corp email` | Корпоративный email с максимумом buzzwords |
| `fuck corp review` | Performance review |

## Конфигурация

При первом запуске создаётся файл `~/.config/fuck/config.toml` со всеми фразами по умолчанию.

Отредактируй его, чтобы добавить свои оправдания, диагнозы, цитаты и прочий контент:

```toml
[it]
excuses = [
    "Мой кот сидел на клавиатуре.",
    "Я изучал документацию. Она не помогла.",
    # добавь своё...
]

[off]
rude = [
    "Иди нахер отсюда.",
    # ...
]

[sv]
hooli_quotes = [
    "Мы делаем мир лучше через платформы изменения масштаба.",
    # ...
]

# и так далее для всех секций
```

Если конфиг повреждён — программа предупредит и использует встроенные значения.
